라이딩을 콘텐츠로

촬영한 영상과 라이딩 데이터만 넣으면, 나머지는 RideLens가.

🏁

GPX · FIT 오버레이

가민·와후·브라이튼 등 모든 사이클 컴퓨터의 FIT, 스트라바 GPX로 속도·경사·파워·심박·케이던스를 영상에 합성.

🎯

자동 싱크 · 지연 보정

영상 움직임을 분석해 GPX와 자동 정렬. 정지·출발·경사의 GPS 지연을 보정하고 터널 구간까지 자동 감지.

✂️

컷 편집 · 자막 · 음악

구간 삭제·속도 조절, 자막과 TTS, CC0 배경음악까지. 편집부터 내보내기까지 한 앱에서.

⭐

하이라이트 자동 추출

몇 시간짜리 라이딩도 GPX 데이터(속도·경사·파워)로 명장면을 골라 15분 하이라이트를 자동으로 만들어줍니다.

📱

1분 숏츠 원클릭

버튼 하나로 하이라이트 1분을 뽑아 9:16 세로 숏츠로. 제목·자막·브랜드 배너까지 얹어 SNS에 바로.

🚀

4K 하드웨어 가속

VideoToolbox 하드웨어 인코딩으로 4K도 빠르게 내보내기. 원본 화질 그대로 뽑을 수 있습니다.

실제 화면

프로젝트로 관리

라이딩마다 프로젝트로 저장·복원. 영상과 GPX를 끌어다 놓으면 바로 시작되고, 작업은 자동 저장됩니다.

오버레이를 내 마음대로

속도·파워·케이던스·심박·경사·미니맵·고도 그래프 위젯을 자유롭게 배치·크기 조절. 패널형·심플·HUD·숏츠 전용 등 템플릿을 만들고 골라 쓰세요.

한 화면에서 편집까지

오버레이 미리보기, 멀티클립 타임라인, 싱크 보정, 컷·자막이 한 화면에. 프리뷰를 보며 바로 다듬습니다.

숏츠도 간편하게

메인 영상에서 하이라이트 구간만 잡으면 9:16 세로 숏츠로 자동 추출. 제목·자막·브랜드 배너까지 얹어 SNS에 바로 올릴 수 있습니다.

1분 튜토리얼

프로젝트 만들기 → 영상·GPX 불러오기 → 싱크 맞추기까지, 한눈에.

설치는 30초

macOS

① zip 풀기 → ② RideLens Free.app을 응용 프로그램 폴더로 드래그 → ③ 터미널에 아래 한 줄 붙여넣기 — 순서대로 하면 바로 실행됩니다 (최초 1회만).
xattr -cr "/Applications/RideLens Free.app" && open "/Applications/RideLens Free.app"
💡 앱을 다른 폴더에 뒀다면 xattr -cr 까지 입력 후 앱을 터미널 창에 끌어놓으면 경로가 자동 입력돼요.
터미널이 어려우면: 앱 더블클릭 → 경고 닫기 → 시스템 설정 → 개인정보 보호 및 보안 → 아래 화면처럼 "그래도 열기"
⚙️ 개인정보 보호 및 보안 바로 열기 (Mac에서 누르면 설정이 바로 열립니다)

🪟
Windows

Setup.exe를 실행합니다. 파란 "Windows의 PC 보호" 창이 뜨면 "추가 정보"를 먼저 클릭하세요 — 그래야 숨어있던 "실행" 버튼이 나타납니다 (최초 1회). 아직 베타라 코드 서명 인증서가 없어서 뜨는 정상적인 경고이며, 앱에 문제가 있는 게 아닙니다.← "추가 정보" 클릭 전에는 "실행 안 함"만 보이고, 클릭하면 "실행" 버튼이 나타납니다 →

💻 권장 사양
Windows 10/11 (64bit) · macOS (Apple Silicon) · RAM 8GB 이상(4K는 16GB 권장) · SSD 권장
엔비디아·인텔·AMD 하드웨어 인코더가 있으면 내보내기가 빠릅니다. 없어도 소프트웨어로 동작하며, 저사양이면 출력 화질을 1080p로 두시길 권장합니다.

⚠️ RideLens는 아직 베타라 유료 개발자 인증서(macOS 공증·Windows 코드 서명)를 붙이기 전입니다. 그래서 macOS는 "손상됨", Windows는 "PC 보호" 경고가 뜨는데 — 위 방법으로 최초 1회만 열면 이후엔 정상 실행됩니다. 정식 서명은 준비 중이에요. 문제가 있으면 꼭 제보해주세요!
