# RideLens 다운로드

라이딩 영상에 속도·경사·파워·심박 텔레메트리를 입히는 영상 편집기 (GPX/FIT 지원).

## 최신 다운로드
👉 [Releases 페이지에서 받기](https://github.com/jaden2dev/ridelens-releases/releases/latest)

- **macOS (Apple Silicon)**: `RideLens-Free-mac-arm64.zip`
- Windows: 준비 중

## 설치 (macOS)
1. zip 압축 해제 후 `RideLens Free.app`을 응용 프로그램 폴더로 이동
2. 처음 실행 시 앱 우클릭 → "열기" → "열기" (서명 전 베타)

문의·건의: [ridy.kr](https://ridy.kr)
